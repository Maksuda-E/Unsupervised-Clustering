# Mall Customer Segmentation Project

## Overview

This project converts the unsupervised clustering notebook into a modular Python project and a Streamlit app.

The notebook first shows customer groups clearly with 5 clusters using Annual Income and Spending Score.
That simple 5 cluster meaning is used in this app for easy explanation.

## Files

app.py contains the Streamlit app.
main.py runs the training pipeline.
src contains the modular Python files.
artifacts stores saved model files and outputs.
logs stores the project log file.
